### Function
* 검색된 데이터를 지도에 `PIN으로 표시`하여 쉽게 알아볼 수 있도록 구현
* 검색 후 `단계별 분류`를 통한 세부 검색 기능 구현
* 디자인적 요소를 위한 Bootstrap의 Navbar 기능 사용
* Bootstrap의 Popovers을 통해 우측 상단에 간단한 사용법 제공
